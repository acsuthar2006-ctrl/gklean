# gklean

**Version control without the fear.**

Git is powerful but intimidating. `gklean` is a command-line tool that wraps complex Git workflows into simple, intuitive commands. Designed for beginners, it prevents common mistakes and provides friendly, human-readable feedback.

**Core Features:**
- `gklean save`: Combines add and commit safely.
- `gklean sync`: Handles pulling and pushing updates.
- `gklean undo`: Fix mistakes without losing work.
- `gklean status`: Tells you exactly what's happening in plain English.
